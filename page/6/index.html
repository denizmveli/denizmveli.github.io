Vodafone Usage v1.2

December 13, 2011
An update to both Plus and Free versions has been released. The key new feature released only for Plus version users, are warning notifications with customisable warning levels. Users now have the ability via the settings menu to enable notifications. These notifications will appear in your notifications area when usage data is updated (either via a background update or the application itself) and the percentage used is equal to or over the selected warning level.
