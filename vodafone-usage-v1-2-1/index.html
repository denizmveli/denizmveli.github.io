Vodafone Usage v1.2.1

December 13, 2011

RELEASE

A minor update has been released tonight for both the free and plus version of Vodafone Usage. All current users are recommended to install the update.

The release includes a change to HTTP protocol settings in an attempt to improve connection stability. The myvodafone.com.au website, from where the application sources it’s usage data, is fairly unstable, however some minor tweaks have been made to try to improve things on our side.

Below is a list of common error messages from the app and what you can do to resolve it.

  1. Connection Error

This most likely points to the myvodafone.com.au web service being down, for either maintenance, or just because Vodafone can’t keep it consistently accessible. If it is your first time using the app and you see this error, please try accessing the myvodafone web site from the browser on your android mobile. This will confirm you have connectivity to the required site.

  1. No data found on page to process

This error normally means that the application was unable to successfully login to the Vodafone web site before retrieving your usage data. Please verify your username and password if they have changed, or it is your first time using the app. This issue can also be caused by a slow connection, or a connection dropping out. If you are using a 3g connection try the app via WIFI if possibly, to see if connection speed is the issue.

Additionally if you have logged into the myvodafone website via another browser, either on your PC, Tablet, TV, Fridge, etc, please ensure you log off the site. Occasionally having multiple active logins on the vodafone site causes issues.

  1. Parse Error : Usage data not delivered.

This error occurs when the application has logged in successfully however the Vodafone website has not delivered the usage data page correctly. This is a Vodafone website error and something we can not cater for in the application. The best response is to try the application again, maybe after a few minutes, which usually fixes this issue for most users. If the error consistently reappears contact The Modern Ink for further support.

We hope this helps our users to get the most out of Vodafone Usage.

Any questions please don’t hesitate to contact us via email or twitter.
