Vodafone Usage 1.4 Released

December 13, 2011

Tonight we’ve released a long awaited update to Vodafone Usage and Vodafone Usage plus.

Changes include maintenance to the connection library to fix errors and improve stability when connecting to Vodafone. This should fix a lot of connection errors people were experiencing as well as the incorrect error stating that the username and password were wrong.

Also a new feature has been added (only for Plus users) allowing for widget look and feel customisation, including the background (dark or transparent) and the text colour.

This should satisfy those minimalistic home screen lovers out there.

You can get the update on the Android Market now (or when it updates……)

https://market.android.com/developer?pub=The+Modern+Ink
