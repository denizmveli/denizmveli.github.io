Vodafone Usage v1.2

December 13, 2011

An update to both Plus and Free versions has been released.

The key new feature released only for Plus version users, are warning notifications with customisable warning levels. Users now have the ability via the settings menu to enable notifications. These notifications will appear in your notifications area when usage data is updated (either via a background update or the application itself) and the percentage used is equal to or over the selected warning level. Users can also choose to be notified only once, or leave warning notifications on for every update.

We recommend setting this to either 85 or 90 percent so you are aware as soon as you are getting close to reaching your bundles allowance.

Another change available to both versions is an improvement to how automatic updates work. Automatic background updates can now be disabled and enabled, via the settings screen, and are not dependent on whether a widget exists on your home screen or not.

Previously a widget needed to be added to enable automatic updates and there was no way to stop automatic updates (i.e. do manual updates) without removing the widget from the screen. The new settings are clearer and give the user more control.

Oh and some other fixes were included for WVGA screen users (Nexus One, Desire, etc). Widget layout clipping has been fixed and most icon resolutions have been improved for these device.

We look forward to hearing your feedback on this release. Any questions or issues please drop us an email or you can tweet us @themodernink.
