Vodafone Usage gets Prepaid

December 13, 2011

If you are a Vodafone Prepaid user with an Android phone, then the day as come. Tonight we have released the initial version of Vodafone Usage with support for Prepaid recharges.

Initial support for Prepaid is for “Flexi Cap” and “All-Time” recharges only. As this is a brand new feature please report any issues via email so we can fix them quick.

Be sure to set your account type to Prepaid in the settings and you are ready to go.

You can head over to the Android Market to download it right now.
