Vodafone Usage Australia Released

December 13, 2011

The Modern Ink is pleased to announce the release of its first Android application.

Vodafone Usage Australia

The essential application Vodafone Android customers have been waiting so long for is now available for download from the Android Market. Managing your mobile phone usage is now simple and painless. Bills will be reduced, usage will go unwasted, Vodafone will get mad :)

Download it now! And don’t forget to share this with your friends.

You can find us on the Android Market (just search Vodafone Usage)
or scan the 2D barcode below on your phone

Version 1.0.0 - Supports all Android phones.
Android version 1.6+

Features include:
Secure and easy access to Vodafone postpaid usage data.
Support for all known postpaid plan bundles including new Vodafone to Vodafone free minutes.
Widget with configurable update interval for simple background updates to always keep you up to date.
Status icons to help manage usage evenly.

  • Green - current usage rate is on track to finish under your monthly allowance.
  • Yellow - current usage rate is on track to finish close to or slightly over your monthly allowance.
  • Red - continuing current usage rates will go over your monthly allowance, or you are over 95% usage.

Features to be released in the next major release:
Configurable 1x1 widget.
Warning notifications when usage exceeds users configured warning level.

The application is ad supported. It has many features and there are many more to come. If you don’t like features and don’t like ads, please don’t download the application. Check your usage via the website or sms. It’s free.

If you do like features and want to support Australian Android developers, then please click the ads. You could even make a donation via the paypal link above.

Thanks to everyone who has helped with the development and testing of this application. Please send any feedback or issues to apps@themodernink.com If you’d like to support our developments please make a donation via paypal (link is up the top right).
